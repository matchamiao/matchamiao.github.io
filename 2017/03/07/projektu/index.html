Projekt-U

unbekennt und unglaublich

Ein komplexes geometrisches System, das auf bestimmtes Algorithmus basiert und seiner Anwendung im Breich der Architektur.

Designer: Jiaming Chen and Wenyu Zhou

Location: University Kassel Germany







Das komplexe geometrische System

Unser Hauptsystem hat 3 unterschiedliche Subsysteme. Jedes System hat eine enge Verbindung miteinander. Gleichzeitig hat jedes System eigenen Parameter und versichdene Gewichtung des Einfluss auf dem ganzen System.


  • Das Oberflächesystem

  • Das räumlich System

  • Das Volumenssystem

Das Algorithmus

Die Logik von des Algorithmus ist die Verteilung der Oberfläsche. Je mehr Mals die Fläche verteilen, desto mehr Einzielheiten bekommen wir.






Die Säule

Zuerst haben wir eine Säule erschafft. Das ist auch ein sinvoller Versuch, weil die Säule das wichtigsten Element in Bezug von Architerktur in menschlicher Geschichte ist.






Eine poetisch Existenz

Mit Hilf des Algorithmus haben wir eine möbelbezogene Möglichkeit erschafft. Das ist eine poetisch Existenz mit dramatische Räume und interessante Einzielheiten.
